## Power BI Dashboard Project

This repository contains a **Sales Dashboard** created using Microsoft Power BI Desktop.  
The project demonstrates skills in **data cleaning, data modeling, and interactive visualization design**.  

### Key Features:
- Connected and transformed raw sales data for analysis  
- Built interactive visuals including bar charts, line charts, and KPI cards  
- Implemented slicers and filters for dynamic data exploration  
- Designed clean, user-friendly layout for better insights  

### Purpose:
This project was created as part of my learning in **data analytics and business intelligence**.  
It highlights my ability to turn raw data into meaningful insights through Power BI dashboards.
