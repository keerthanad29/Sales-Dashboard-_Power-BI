## 📌 Project Overview

This project demonstrates end-to-end **Business Intelligence** skills — from connecting and cleaning raw sales data to designing an interactive, recruiter-ready dashboard in Microsoft Power BI Desktop.

## ✨ Key Features

- 📊 **KPI Cards** — Total Sales, Revenue, Orders at a glance
- 📅 **Time-series Analysis** — Monthly and yearly sales trends
- 🌍 **Regional Breakdown** — Sales performance by region/category
- 🎛️ **Interactive Slicers** — Filter by date, region, product dynamically
- 🧹 **Data Cleaning** — Handled missing values and transformed raw data using Power Query

---

## 📈 Key Insights from the Dashboard

- 🏆 Top-performing product categories identified by revenue contribution
- 📅 Sales peak seasons visible through time-series trend analysis
- 🌍 Regional performance gaps highlighted for strategic decision-making
- 📉 Underperforming segments flagged using conditional formatting

## ⚙️ How to Open

```
1. Download and install Power BI Desktop (free) from https://powerbi.microsoft.com
2. Clone or download this repository
3. Open "Sales Dashboard.pbix" in Power BI Desktop
4. Explore the interactive dashboard!
```
